Mas afinal, o que são apps mobile?

Os apps mobile são instaladas diretamente através de lojas de aplicativos ddos aparelhos móveis e funcionam como qualquer outro aplicativo para computador, só que são desenvolvidas para dispositivos móveis, tendo em conta os diferentes sistemas operacionais, para funcionarem têm de ser compatíveis com o aparelho, pensadas para o tipo de interface e dimensões de ecrã destes dispositivos.
